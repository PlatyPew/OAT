OAK Amazing Market REST-API

List items in store

List all items in the store and returns as JSON Object

GET /api/market/store/get
OAK: OAK_TOKEN
{ "response": Object }

List items in cart

List all items in the cart and returns as JSON Object

GET /api/market/cart/get
OAK: OAK_TOKEN
{ "response": Object }

Set items in cart

Set the amount in the cart and return cart as JSON Object

GET /api/market/cart/set
OAK: OAK_TOKEN
{ ...items: item_number }
{ "response": Object }

Buy items in cart

Buy from items in cart and return JSON object of items bought

GET /api/market/store/buy
OAK: OAK_TOKEN
{ "response": Object }

Restock items in store

Set the amount in the inventory and return as JSON Object

GET /api/market/store/restock
OAK: OAK_TOKEN
{ ...items: item_number }
{ "response": Object }
